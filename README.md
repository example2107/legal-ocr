# ЮрДок — Распознавание юридических документов

Веб-приложение для распознавания текста с юридических документов на русском языке с функцией обезличивания персональных данных.

## Возможности

- 📄 Загрузка изображений (JPG, PNG, WEBP) и PDF-файлов (многостраничных)
- 🔍 Распознавание текста через Claude API (Sonnet)
- 📋 Сохранение структуры документа
- ⚠️ Выделение неточно распознанных мест
- 👤 Автоматическое обнаружение персональных данных
- 🔒 Обезличивание по клику (ФИО → буква, адрес → [адрес])
- 📥 Копирование и скачивание результата

## Публикация на Vercel

### Вариант 1: через GitHub (рекомендуется)

1. Загрузите папку проекта на GitHub (создайте репозиторий)
2. Зайдите на [vercel.com](https://vercel.com) и войдите через GitHub
3. Нажмите "Add New Project" → выберите ваш репозиторий
4. Настройки оставьте по умолчанию (Vercel сам всё определит)
5. Нажмите "Deploy"

### Вариант 2: через Vercel CLI

```bash
npm install -g vercel
cd legal-ocr
npm install
vercel
```

## Локальный запуск

```bash
npm install
npm start
```

Откройте http://localhost:3000

## Безопасность

- API ключ вводится вручную и **нигде не сохраняется**
- Документы обрабатываются только в браузере
- Никакие данные не передаются на сторонние серверы (кроме Anthropic API)
