# ЮрДок — Распознавание юридических документов

Веб-приложение для распознавания и обезличивания юридических документов на русском языке с поддержкой нескольких ИИ-провайдеров.

## Возможности

### Распознавание
- 📄 Загрузка изображений (JPG, PNG, WEBP) и PDF-файлов (многостраничных, любой размер)
- 📋 Вставка готового цифрового текста для обезличивания
- 🤖 Поддержка трёх ИИ-провайдеров: Claude (Anthropic), ChatGPT (OpenAI), Gemini (Google)
- ⚠️ Автоматическое выделение неточно распознанных мест (двойное оранжевое подчёркивание)
- 🔍 Проверка качества распознавания — поиск OCR-артефактов и семантических ошибок
- 🏗️ Сохранение структуры документа (заголовки, выравнивание, двухколоночные строки)

### Редактор
- ✏️ Встроенный rich-text редактор с форматированием (жирный, курсив, выравнивание, списки, отступы)
- 🖱️ Правый клик на неточном фрагменте → меню «Исправлено — снять выделение»
- Times New Roman 14pt — шрифт редактора совпадает с PDF/DOCX для точного переноса строк
- Выравнивание по ширине по умолчанию

### Персональные данные
- 👤 Автоматическое обнаружение ФИО, адресов, телефонов, паспортных данных и др.
- 🔒 Обезличивание кликом: частные лица → А., Б., В.; профессиональные участники → [ФИО 1], [ФИО 2]
- Категории: частные лица (обвиняемые, свидетели) и профессиональные участники (судьи, следователи)
- Групповое скрытие/показ по категориям

### Сохранение и экспорт
- 💾 Сохранение в историю браузера (localStorage, до 50 документов)
- ⬇️ Скачивание в PDF (через диалог печати браузера)
- ⬇️ Скачивание в DOCX (Times New Roman 14pt, поля A4 20/25мм)
- ⚠️ Предупреждение при сохранении/скачивании если остались непроверенные фрагменты

### Безопасность
- API-ключ не сохраняется — только в памяти текущей сессии
- Документы обрабатываются только в браузере, на серверы не передаются
- История хранится только в localStorage вашего браузера

## Деплой на Vercel

### Через GitHub (рекомендуется)
1. Загрузите папку проекта на GitHub
2. Зайдите на vercel.com → Add New Project → выберите репозиторий
3. Нажмите Deploy

### Через Vercel CLI
```bash
npm install -g vercel
npm install
vercel
```

### Локальный запуск
```bash
npm install
npm start
```

## Структура проекта
```
src/
├── App.js              # Главный компонент, вся логика
├── App.css             # Стили интерфейса
├── components/
│   ├── RichEditor.js   # Редактор + аннотации ПД + контекстное меню
│   └── RichEditor.css  # Стили редактора
└── utils/
    ├── claudeApi.js    # API-клиенты (Claude, OpenAI, Gemini)
    ├── history.js      # История документов (localStorage)
    └── pdfUtils.js     # Конвертация PDF → изображения
```

## Обновление приложения
1. Внесите изменения в файлы
2. Сделайте commit в GitHub Desktop
3. Нажмите Push origin
4. Vercel автоматически пересоберёт (~20 сек)
